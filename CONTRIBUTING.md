# Contributing

Obrigado por contribuir.

## Requisitos
- PHP 8.x
- Servidor web local
- API Laravel disponivel

## Fluxo de Trabalho
1. Crie uma branch `feature/*` ou `fix/*`.
2. Faça commits pequenos e claros.
3. Atualize documentacao quando necessario.
4. Abra um pull request para `main`.

## Padroes
- Evite commits com arquivos gerados.
- Nao inclua credenciais.
- Mantenha o estilo de codigo existente.

## Testes Basicos
1. Login funcionando.
2. Navegacao por views.
3. Modulos criticos carregando dados.
